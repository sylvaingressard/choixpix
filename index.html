PIX : Choisissez les sujets!

Commencez!

Comment utiliser l'outil?

Filtrez!

Le référentiel PIX vous est proposé de façon à être parcouru compétences par compétences.

La barre de gauche vous permet de les filtrer et de les trier. Lorsque vous sélectionnez une compétence, les sujets correspondants s'affichent au centre.

Choisissez!

Pour ajouter un sujet à votre liste de priorités, cliquez simplement sur le bouton ajouter.

Le sujet est ajouté à votre liste de choix:

Exportez!

Le bouton exporter vous permet d'enregistrer vos choix dans un fichier que vous pouvez ouvrir avec Excel.

Vous pouvez à partir de ce ficher recharger vos choix pour les modifier ultérieurement
